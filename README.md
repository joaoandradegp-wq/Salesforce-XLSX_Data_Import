## 📌 Sobre

O **Salesforce XLSX Data Import** foi desenvolvido para facilitar operações com dados no Salesforce, especialmente em ambientes do mercado de **Mobility**, atuando diretamente com registros do objeto **Opportunity** e seus vínculos com **Account** e **Asset**.

A ferramenta converte arquivos `.xlsx` para um formato compatível com o **Salesforce Inspector**, eliminando ajustes manuais e reduzindo erros durante o processo de importação.

---

## ⚙️ Como funciona

- Selecione seu arquivo **.xlsx**  
- O programa realiza a estruturação automatica por objeto: Account, Opportunity e Asset. 
- Utilize o resultado para ser colado diretamente no **Data Import** 

Simples, rápido e focado em produtividade.

Obs: Regras definidas para o mercado de Mobility.

---

## 🔗 Aplicação complementar

Foi incorporado um segundo utilitário com funcionalidade independente:

### 🧠 Opportunity SOQL

Responsável por gerar consultas **SOQL** otimizadas para o objeto **Opportunity**, preparadas para execução no **Data Export**.

### 🚀 Funcionalidades

- Geração automática de SOQL para o objeto Opportunity baseado nos ID´s de Oportunidade copiados.
- Definição manual de retorno da quantidade de registros possível para a ORG.
- Ajuste automático sobre a quantidade baseado nos limites de retorno de registros do Salesforce.  
- Divisão da consulta em múltiplas partes, caso necessário.

O programa controla a quantidade de registros retornados, evitando erros devido ao limite de retorno do Salesforce em uma única requisição.

---

## 🎯 Quando usar

- Importação de dados em massa no Salesforce (Opportunity)  
- Ajuste de planilhas para compatibilidade com o Inspector  
- Extração de grandes volumes de dados via SOQL  
- Operações em ambientes do segmento de Mobility
